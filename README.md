Todo Project
This is a full-stack Todo Project with a frontend and a backend.

📁 Project Structure
todo project/
├── todolist/ # Frontend
└── todo-backend/ # Backend

🛠️ Technologies Used
Frontend: React, HTML, CSS, JavaScript
Backend: Node.js, Express
Database: MongoDB

🚀 How to Run
1. Clone the repository
git clone https://github.com/RishithaReddyNalla/task-manager.git
cd todo-project

2. Start frontend
cd todolist
npm install
npm start

3. Start backend
cd todo-backend
npm install
npm start

✨ Features
Add new tasks

Update tasks

Delete tasks

View all tasks
